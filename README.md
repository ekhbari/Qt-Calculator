# Qt Calc (Arabic)

## Features
- Basic ops: `+`, `-`, `*`, `/`
- Clear btn (C), Equals btn (=)
- Error msg on div/0
- Clean UI via Qt Designer

## Files
- `main.cpp` – app entry  
- `mainwindow.h/.cpp` – logic & slots  
- `mainwindow.ui` – UI layout  
- `.pro` – Qt project file

## Run
1. Open `.pro` in Qt Creator  
2. Build & Run

## Req
- Qt 5+ & Qt Creator
